This is a software tool automatically "aging" Minecraft worlds ("save files"). Food spoils, tools break, walls crumble, buildings fall down.

Use at your own risk!
Created by David Stark: http://www.zarkonnen.com/
Source code available at: https://github.com/Zarkonnen/Ager
